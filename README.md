# 🎯 ThumbAI - Viral YouTube Thumbnail Generator

> AI-powered thumbnail generator that creates click-worthy YouTube thumbnails in seconds. Built for creators worldwide, optimized for US & Europe markets.

## 🚀 Features

- **5-Step Smart Wizard**: Answer quick questions about your niche, topic, emotion, and style
- **AI-Powered Generation**: Powered by Nano Banana 2 trained on 500K+ viral thumbnails
- **Canvas-Based Rendering**: Real-time thumbnail preview with customizable text and colors
- **Multiple Styles**: Bold & Loud, Minimal, Face Reaction, Listicle, VS, Story templates
- **Emotion Targeting**: Curiosity, Shock, FOMO, Inspiration, Fear, Humor, Trust, Excitement
- **HD Export**: Download at 1280×720 (4K in Pro version)
- **Subscription Model Ready**: Free, Pro ($19/mo), Agency ($99/mo) tiers

## 📁 Tech Stack

- **Frontend**: Pure HTML5, CSS3, Vanilla JavaScript
- **Design**: Dark sleek UI with gradient accents
- **Canvas API**: For real-time thumbnail rendering
- **No Dependencies**: Zero build tools, works out of the box

## 🛠️ Installation

### Local Development

```bash
# Clone the repo
git clone https://github.com/aadi4frnzzz-crypto/thumbai-viral-thumbnail-generator-2026-04-25.git
cd thumbai-viral-thumbnail-generator-2026-04-25

# Open in browser (no build needed!)
open index.html
# or use a local server
python -m http.server 8000
```

### Deploy to Production

**Vercel:**
```bash
vercel --prod
```

**Netlify:**
```bash
netlify deploy --prod --dir .
```

**GitHub Pages:**
1. Go to Settings → Pages
2. Source: Deploy from branch `main`
3. Live at: `https://aadi4frnzzz-crypto.github.io/thumbai-viral-thumbnail-generator-2026-04-25/`
